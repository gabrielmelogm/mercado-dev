# Mercado dev

## Description

A project to study microservices that replicates the mercado livre platform.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Running only app

```bash
# products
$ pnpm run start:dev products

# orders
$ pnpm run start:dev orders

# payments
$ pnpm run start:dev payments
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
